# Entrenador de Ajedrez: the WoodPecker Method

Este programa permite hacer un entrenamiento de ajedrez usando el metodo "del pajaro carpintero"

Consiste en establecer un periodo de tiempo, llamado ciclo, en el que se resolverá la mayor cantidad de puzzles
Una vez completado un ciclo, se reducirá el rango de tiempo a la mitad y se repetiran los puzzles hechos.

Es como un repaso espaciado de puzzles de ajedrez

Se usara la lista de puzzles publicada por lichess.org
